GPX Pacer

GPX Pacer is a tool that analyzes GPX files to generate pacing splits for ultra running. Upload a GPX file to calculate splits by distance or waypoints, elevation gain/loss, and optional surface types.

Built on the GPX Pacer project by Zettt. Useful for planning pacing strategies on trail races and ultra marathons.
