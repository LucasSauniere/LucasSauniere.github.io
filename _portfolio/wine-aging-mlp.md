## Overview

A personal side project exploring the application of **machine learning** to an unexpected domain: wine science. The goal is to predict the **quality and aging potential** of wines based on their physicochemical composition, using Multi-Layer Perceptrons (MLPs).

## Approach

- **Dataset:** Wine quality datasets (e.g., from the UCI ML repository) containing features such as acidity, sugar content, alcohol level, pH, sulphates, etc.
- **Model:** Multi-layer perceptron (MLP) with tuned architecture for regression and classification tasks
- **Evaluation:** Cross-validation, confusion matrices, regression metrics (RMSE, R²)

## Key Questions

- Can we predict wine quality (score) from its chemical properties?
- Can aging potential be estimated before the wine is actually aged?
- Which chemical features are most predictive of quality?

## Technologies

- **Python** (scikit-learn, PyTorch, pandas, matplotlib, seaborn)
- **MLPs / feedforward neural networks**
- **Feature importance analysis** (SHAP values, permutation importance)
- Jupyter Notebooks

## What I Learned

This project was a fun way to apply ML to a non-standard scientific domain, reinforcing skills in **data preprocessing**, **hyperparameter tuning**, **model evaluation**, and **feature engineering** outside of a purely physical/optical context.
