## Overview

Beyond my main PhD research, I explore **computer vision** and **deep learning** applied to various image analysis tasks. These projects help me develop a broad skill set across different domains of AI and image processing.

## Projects

### Object Detection & Segmentation
- Experimenting with YOLO, Faster R-CNN, and Mask R-CNN architectures
- Fine-tuning pre-trained models on domain-specific datasets
- Performance benchmarking and visualization

### Image Classification
- Building and training CNNs from scratch and via transfer learning
- Applying data augmentation techniques for limited-data regimes
- Multi-class classification pipelines with PyTorch and TensorFlow

### Generative Models
- Experimenting with autoencoders and variational autoencoders (VAEs) for image reconstruction
- Using generative approaches for data augmentation in scientific imaging contexts
- Inspired by work done during the PhD on PSF reconstruction

### Astronomical Image Analysis
- Applying deep learning to data from Kepler/K2 and TESS space telescopes
- Light curve analysis and period detection for exoplanet transit identification
- Aperture photometry pipelines

## Technologies

- **Python** (PyTorch, TensorFlow/Keras, OpenCV, scikit-image)
- **Deep learning** (CNNs, autoencoders, transfer learning, fine-tuning)
- **Astropy** for astronomical image processing
- **Matplotlib, Plotly** for visualization
- Linux / HPC environments, Jupyter Notebooks
